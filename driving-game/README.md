# City Drive - POV Car Game

A simple browser-based driving game where you navigate through a city grid while avoiding traffic.

## 🎮 How to Play

- **Arrow Keys** or **WASD** to drive
- **Space** to brake
- Avoid other cars and stay within the city limits
- Drive faster to increase your score

## 🚀 Play Online

You can play the game immediately by opening `index.html` in any modern web browser.

## 🛠️ Technologies Used

- HTML5 Canvas
- CSS3
- Vanilla JavaScript

## 📝 Game Features

- Top-down perspective with camera following
- Procedurally generated city grid
- AI traffic cars to avoid
- Speed-based scoring system
- Speed lines for POV effect

## 🎯 Game Objective

Drive as long as possible without crashing into other vehicles or leaving the city. Your score increases faster at higher speeds.

## 🔧 Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. Start driving!

## 📱 Compatibility

Works on all modern browsers (Chrome, Firefox, Safari, Edge). For best experience, use a desktop/laptop with keyboard.
